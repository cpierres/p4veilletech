Modules de Spring et Spring Boot pour les microservices

Module Rôle
Spring Boot Starter Web Permet de créer des APIs REST en exposant des endpoints HTTP.
Spring Boot Starter Actuator Fournit des endpoints de monitoring (santé, métriques, etc.) pour gérer chaque microservice.
Spring Cloud Netflix Eureka Met en place un service de découverte pour que les microservices puissent s'enregistrer et se découvrir dynamiquement.
Spring Cloud Gateway Agit comme un point d'entrée unique (API Gateway) pour router les requêtes vers les différents microservices.
Spring Cloud Config Permet une gestion centralisée de la configuration pour tous les microservices.
Spring Security Implémente des mécanismes de sécurité comme l'authentification et l'autorisation pour chaque microservice.
Spring Cloud Sleuth Fournit des outils pour le traçage distribué afin de suivre les requêtes à travers différents microservices.
Spring Boot DevTools Facilite le développement avec un redémarrage automatique et un flux de travail simplifié.
Spring Data Fournit des abstractions pour accéder aux bases de données dans chaque microservice.
Spring Cloud OpenFeign Permet de configurer des clients HTTP pour la communication entre microservices de manière déclarative.
